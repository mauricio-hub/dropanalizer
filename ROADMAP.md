# Proply — Roadmap

## Phase 1 — First working feature (current focus)

Goal: a user can create and see their proposals.

- [x] Project setup (Next.js, Prisma, Clerk, Tailwind)
- [x] DB schema (Proposal, Version, Event)
- [x] Clerk auth (sign in / sign up)
- [x] Design system + UI components
- [x] POST /api/proposals — create proposal
- [x] GET /api/proposals — list proposals
- [ ] Dashboard page — list user's proposals
- [ ] New proposal page — title + brief form
- [ ] Save proposal to DB on submit
